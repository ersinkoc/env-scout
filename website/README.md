# EnvScout Website

This is the official website for EnvScout - a comprehensive environment detection library for JavaScript/TypeScript applications.

## Website Features

- **Modern Design**: Built with Tailwind CSS for a clean, professional look
- **Responsive**: Works perfectly on all devices (mobile, tablet, desktop)
- **Interactive**: Powered by Alpine.js for smooth user interactions
- **Performance**: Optimized for fast loading and great user experience

## Sections

1. **Hero Section**: Eye-catching introduction with live environment detection demo
2. **Features**: Showcase of library capabilities
3. **API Reference**: Complete function documentation
4. **Code Examples**: Real-world usage examples
5. **Statistics**: Version, dependencies, and license info
6. **Call-to-Action**: Quick installation and GitHub links

## Deployment

This website is automatically deployed to GitHub Pages when changes are pushed to the main branch.

**Live URL**: [https://env-scout.oxog.dev](https://env-scout.oxog.dev)

## Local Development

Simply open `index.html` in your browser to view the website locally.

## Technologies Used

- **HTML5**: Semantic markup
- **Tailwind CSS**: Utility-first CSS framework
- **Alpine.js**: Lightweight JavaScript framework
- **Font Awesome**: Icon library
- **Prism.js**: Syntax highlighting for code blocks
